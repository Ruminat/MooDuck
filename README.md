### The MooDuck bot

Tell him how you feel today from 1 to 10

### How to develop

Clone the repo:

```bash
git clone https://github.com/Ruminat/MooDuck.git
```

Install the dependencies with:

```bash
npm install
```

Run in dev mode:

```bash
npm run dev
```

Or build for production:

```bash
npm run build

# then start the build version with
npm run start
```
